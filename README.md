# 📚 Repositorio de Masters Of Code

¡Bienvenido a nuestro repositorio! Aquí documentaremos todo el código, ejercicios, proyectos y evaluaciones desarrolladas a lo largo de las clases del curso. El contenido está organizado por lenguajes de programación.

---

## 🗂️ Estructura del Repositorio

El proyecto está dividido en las siguientes carpetas principales para mantener el orden de las clases:

* **`Java/`**: Contiene los ejercicios y prácticas realizados en Java. Incluye todas las clases vistas hasta la fecha.
* **`Javascript/`**: Directorio dedicado a los ejercicios, scripts y proyectos de las clases de JavaScript.
* **`Python/`**: Contiene los ejercicios y practicas realizadas en python. Incluye todas las clases vistas hasta la fecha.
* **`Juegos/`**: Proyectos basados en juegos.
* **`quiz/`**: quizes de todas las semanas.

> **Nota:** La carpeta **`.idea/`** contiene configuraciones locales del entorno de desarrollo (IDE) y el archivo **`.gitignore`** se encarga de evitar que se suban archivos innecesarios al repositorio.
